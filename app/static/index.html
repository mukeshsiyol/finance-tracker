API Reference
Authentication
POST /auth/register Register new user
POST /auth/login Get JWT token
GET /auth/me Current user profile
Transactions
GET /transactions/ List + filter records
POST /transactions/ Create record · admin
GET /transactions/{id} Get single record
PATCH /transactions/{id} Partial update · admin
DELETE /transactions/{id} Delete record · admin
Analytics
GET /analytics/summary Income / expense / balance
GET /analytics/categories Per-category breakdown · analyst+
GET /analytics/monthly Monthly net totals · analyst+
GET /analytics/dashboard Full dashboard · analyst+
Export
GET /export/csv Download as CSV · analyst+
GET /export/json Download as JSON · analyst+
Users (Admin Only)
GET /users/ List all users
GET /users/{id} Get user by ID
PATCH /users/{id}/role Update role
DELETE /users/{id} Delete user
Quick Start
STEP 01

Register & Login

Create an account, then exchange credentials for a JWT token.

POST /auth/register { "username": "alice", "email": "alice@co.com", "password": "secret123", "role": "admin" } POST /auth/login → "access_token": "eyJ..."
STEP 02

Manage Records

Use the Bearer token to create, view, and filter transactions.

Authorization: Bearer <token> POST /transactions/ { "amount": 45000, "type": "income", "category": "Salary", "date": "2025-04-01" }
STEP 03

Get Analytics

Fetch summaries, breakdowns, and export your data.

GET /analytics/dashboard → summary + categories + monthly totals + recent 10 transactions GET /export/csv?type=expense → transactions.csv
